# Arabic Grammar Cheatsheet

I hope this cheat sheet is useful to whoever uses it. It will double over as my notes, and ideally a useful guide to whoever uses this in the future. 

## Pronouns

 |          | English     | Arabic |
 | -------- | ----------- | ------ |
 | singular | I           | أنا    |
 |          | you (masc)  | أنتَ   |
 |          | you (fem)   | أنتِ   |
 |          | he          | هو     |
 |          | she         | هي     |
 | dual     | we          | نحن    |
 |          | you         | أنتما  |
 |          | they        | هما    |
 | plural   | we          | نحن    |
 |          | you (masc)  | أنتم   |
 |          | you (fem)   | أنتن   |
 |          | they (masc) | هم     |
 |          | they (fem)  | هن     |
